## 四、学习方式建议

### 对比学习法

因为你有 Python 和 Java 基础，建议用对比的方式学习：

```go
// Python: x = 1
// Java:   int x = 1;
// Go:     x := 1  （短变量声明，自动推断类型）

// Python: def add(a, b): return a + b
// Java:   int add(int a, int b) { return a + b; }
// Go:     func add(a, b int) int { return a + b }
```

### 每日学习流程

1. **看概念**（15分钟）：阅读教程对应章节
2. **写代码**（30分钟）：动手实践，不要只看
3. **做对比**（15分钟）：和 Python/Java 对比差异
4. **记笔记**（10分钟）：记录关键差异和易错点
