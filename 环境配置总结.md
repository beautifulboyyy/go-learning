## 二、环境变量修改

### 修改内容

**操作：** 将 Go 的 bin 目录添加到用户 PATH 环境变量

```
新增路径：C:\Program Files\Go\bin
```

**执行的命令：**

```powershell
[Environment]::SetEnvironmentVariable('PATH', [Environment]::GetEnvironmentVariable('PATH', 'User') + ';C:\Program Files\Go\bin', 'User')
```

**修改位置：** Windows 用户环境变量（User PATH）

**⚠️ 注意：** 需要重新打开终端才能生效。当前会话通过 `export PATH` 临时生效。
